# Automated Halloween Candy Bowl

## Overview
Interactive embedded system with sensor + RF-based automation. This project creates an engaging Halloween experience by detecting user interaction and triggering automated responses through wireless communication and programmed behaviors.

## Features
- Sensor-based interaction detection  
- Automated event-driven responses  
- RF communication for remote triggering  
- Customizable behavior logic  
- Expandable for additional effects  

## Technologies
- Arduino (C/C++)  
- Proximity / motion sensors  
- RF communication modules  
- Output components (lights, sound, or motors)
